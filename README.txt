  双击 install.bat，等待所有包安装完成。

  │  本批次包含:                 │
  │  ✔ watchdog      (文件监听)  │

⚙️ 第二步：运行程序

  双击 SmartFileOrganizer.py

  ⚠ 首次运行会自动从 Hugging Face 下载
    "paraphrase-multilingual-MiniLM-L12-v2" 模型
    （约 80MB），请保持网络畅通。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 使用说明

  1. 打开程序后，配置两个路径：
     - 「监视文件夹」→ 放待分类文件的地方
     - 「目标文件夹」→ 已经分类好的文件夹（学习范本）
  
  2. 点击「🧠 训练系统」扫描目标文件夹，训练模型
  
  3. 点击「🚀 启动服务」开始自动分类

  4. 未分类的文件会自动移到「未分类文件」文件夹
     在「待处理文件」区域手动选择目标目录即可
