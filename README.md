# SmartFileOrganizer 智能文件分类器 🦝

基于语义理解的智能文件分类工具。通过机器学习自动学习你的文件归类习惯，实现文件的智能分类整理。

## ✨ 功能

- ✅ **语义分类** — 基于 sentence-transformers 多语言模型理解文件内容
- ✅ **增量学习** — 你的每次手动纠正都会被记住，越用越准
- ✅ **实时监听** — 自动监控文件夹，新文件到达即处理
- ✅ **文件去重** — SHA256 哈希检测，避免重复存储
- ✅ **GUI 界面** — 全 Tkinter 图形界面，操作直观
- ✅ **训练系统** — 可从已有文件夹结构学习分类规则

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 运行
python main.py
```

首次运行会自动下载中文语义模型（约 80MB），请保持网络畅通。
